# Geofront-Server
Geofront Server written in pure C with Sockets and pThreads 

## Building:

### Requirements:
* gcc
* make
* pthreads
### Compilation:
* `git clone 'https://github.com/DavidoRotho/Geofront-Server.git'`
* `make`
### Installaion (optional):
* `make install` 
 
 
### Running:
* If installed:
  * `geofront_serv`
* If not installed, run from the build folder:
  * `./geofront_serv`
